System Integration

  1. Home
  2. System Integration

Software development services for integrating equipment, systems and subsystems at an interface level:

  • Sensors
  • Processors
  • Data Links and Communications


The range of equipment, systems and subsystems that we can integrate with include:

  • Radars, EO/IR payloads, LIDAR, GPS/INS
  • Cockpit Control and Display Units (CDUs)
  • Satellite communication equipment (L-band, Ku/Ka-Band)
  • Radio communication equipment (V/UHF band)

Given a set of equipment Interface Control Documents (ICDs), we can write the necessary software code to integrate equipment, systems and subsystems at an interface level.

A typical software-based interface processor will consist of:

  • Protocol parser (binary, or text formatted)
  • Interface controller
  • Data transformer
  • Data publisher
