Visual Object Classification and Detection at the Edge

  1. Home

Visual object classification and detection can be performed at the edge with recent advances in CPU and GPU technologies.


The choice of AI compute can be determined by the use-case, size, weight, power and performance constraints of the platform operating at the edge.
